EC Site — Guide học lập trình theo chương

Mỗi chương dạy bạn tự code dựa trên bộ tài liệu thiết kế trong docs/ (21 văn bản EC-000〜EC-020). Code mẫu chỉ có cho phần "chính" — phần còn lại là 📝 課題 (bài tập), bạn tự làm rồi đối chiếu sau. Roadmap dưới đây nhóm 31 chức năng (07_機能一覧.md) theo 11 nhóm nghiệp vụ (CAT-001〜011), theo đúng thứ tự phụ thuộc: Auth → Catalog → Cart/Order → Payment → Inventory → Admin.
