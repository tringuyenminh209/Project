# 1. 開発目的

本システムは、社員の出退勤情報、休暇申請および勤務状況を効率的に管理することを目的として開発する。

従来の紙やExcelによる勤怠管理をシステム化することで、以下を実現する。

* 勤怠管理業務の効率化
* 人的ミスの削減
* リアルタイムな勤務状況の把握
* 勤怠データの一元管理
* 月次集計業務の自動化

本プロジェクトでは、

* 打刻

* CSV出力
